## 设计决策

### 决策 1：为什么用 Rust？

| 考量因素 | Rust 优势 |
|---------|----------|
| **性能** | 零成本抽象，文本处理高效 |
| **安全性** | 编译期类型检查 |
| **并发** | Tokio 异步运行时 |

### 决策 2：为什么群聊模式？

| 考量因素 | 群聊模式优势 |
|---------|-------------|
| **透明性** | 所有 Agent 思考可见 |
| **实时性** | SSE 流式输出 |
| **可追溯** | 完整对话历史 |

### 决策 3：为什么唯一写入者？

| 考量因素 | 唯一写入者优势 |
|---------|---------------|
| **一致性** | 风格统一 |
| **责任** | 清晰可追溯 |
| **冲突** | 集中处理 |
